1. ER Diagram

แผนภาพความสัมพันธ์ของตารางสำหรับระบบแจ้งแบบ กวภ.01 - กวภ.05 โดยใช้ kwp_form_submissions เป็นตารางหัวฟอร์มกลาง และแยกตารางรายละเอียดตามประเภทแบบฟอร์ม

PK = Primary Key FK = Foreign Key NOT NULL = Required สีส้ม = กวภ.01 สีม่วง = กวภ.03 สีเขียว = กวภ.05
เส้นทึบคือความสัมพันธ์ผ่าน submission_id ไปยัง kwp_form_submissions.id ส่วน kwp_form_attachments สามารถแนบไฟล์ระดับฟอร์มโดยตรง หรือชี้ไปยังรายการย่อยแบบ optional ผ่าน related_table และ related_id
