# Backend Documentation Archive

โฟลเดอร์นี้เก็บเฉพาะเอกสาร backend ที่มีคุณค่าทางประวัติและไม่ซ้ำกับเอกสาร active

เอกสารใต้ `archive/`:

- ไม่ใช่ API contract หรือ implementation reference ปัจจุบัน
- ต้องระบุว่า archived และลิงก์ replacement เมื่อมีเอกสารใหม่แทน
- ต้องไม่ถูกใช้เป็นปลายทางจากคู่มือหรือ API index ปัจจุบัน

ไฟล์ที่เป็นเพียงสำเนาของเอกสาร canonical ไม่ต้องเก็บที่นี่ เพราะสามารถดูย้อนหลังได้จาก Git history
