สรุป Flow ที่เข้าใจ

  • โรงงาน Login ผ่าน iSingleForm เลือกรายงานค่าความคลาดเคลื่อน เลือกจุดตรวจวัด กรอกแบบและแนบไฟล์
  • ระบบตรวจเบื้องต้น: ระยะเวลา 30 วัน, แนบไฟล์ครบ 3 รายการ, ขนาดไฟล์ไม่เกิน 2 MB ถ้าไม่ผ่านให้แก้ไขก่อนส่ง
  • หลังส่งรายงาน ระบบมอบหมายงานอัตโนมัติให้เจ้าหน้าที่ตามพื้นที่ ส่วนกลาง/ภูมิภาค
  • เจ้าหน้าที่ตรวจความถูกต้อง ถ้าไม่ถูกต้องส่งกลับโรงงานแก้ไข ถ้าถูกต้องอนุมัติผลและส่งต่อขั้นทบทวน/อนุมัติ
  • ผู้อำนวยการกดรับงาน พิจารณาอนุมัติ หากไม่อนุมัติส่งกลับทบทวน หากอนุมัติส่งแบบแจ้งผลให้โรงงาน
  • โรงงานเห็นแบบแจ้งผลเป็น PDF และสามารถดาวน์โหลด/พิมพ์ได้
