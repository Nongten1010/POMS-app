# Overview

## เมนูขอเชื่อมต่อ

เมนู "ขอเชื่อมต่อ" เป็นเมนูสำหรับให้ผู้ประกอบการส่งแบบฟอร์มคำขอเพื่อทำธุรกรรมเกี่ยวกับการเชื่อมต่อข้อมูลเข้าสู่ระบบ POMS ได้แก่ การเชื่อมต่อใหม่ การเพิ่มจุดตรวจวัด และการเพิ่มพารามิเตอร์

เมื่อผู้ประกอบการส่งแบบฟอร์มคำขอแล้ว เจ้าหน้าที่ที่มีสิทธิ์จะดำเนินการตรวจสอบ แก้ไข อนุมัติ อนุญาต หรือยกเลิกคำขอตามขั้นตอนของระบบ เมื่อคำขอผ่านขั้นตอนที่กำหนด ระบบ POMS จะได้รับข้อมูลจากจุดตรวจวัดและนำมาแสดงผลในระบบ

### สิทธิ์การใช้งาน

ผู้ที่มีสิทธิ์ใช้งานเมนู "ขอเชื่อมต่อ" แบ่งเป็น 3 ประเภท

#### View

สิทธิ์สำหรับแสดงข้อมูลเท่านั้น โดยแต่ละผู้ใช้งานจะมี scope การมองเห็นข้อมูลดังนี้

| Scope | การมองเห็นข้อมูล |
| --- | --- |
| `ALL` | เห็นข้อมูลทั้งหมด |
| `IN_PROVINCE` | เห็นเฉพาะจังหวัดที่ได้รับสิทธิ์ |
| `IN_ESTATE` | เห็นเฉพาะนิคมที่ได้รับสิทธิ์ |
| `OWN_FACTORY` | เห็นเฉพาะโรงงานของตัวเอง |
| `null` | ไม่อนุญาตให้เข้าถึงเมนูนี้ |

#### Edit

สิทธิ์สำหรับทำธุรกรรมต่างๆ ในเมนู "ขอเชื่อมต่อ" เช่น กรอก แก้ไข และส่งแบบฟอร์มคำขอ

#### Approve

สิทธิ์สำหรับตรวจ แก้ไข อนุมัติ อนุญาต และยกเลิกแบบฟอร์มคำขอ

### ส่วนประกอบของเมนู

เมนู "ขอเชื่อมต่อ" มีเมนูย่อย 2 เมนู

1. รายชื่อโรงงาน
2. รายการคำขอ

### เมนูย่อยรายชื่อโรงงาน

เมนูย่อย "รายชื่อโรงงาน" แสดงผลเป็นตาราง โดยประกอบด้วยคอลัมน์ดังนี้

| คอลัมน์ |
| --- |
| ชื่อโรงงาน/บริษัท |
| เลขทะเบียนโรงงาน (ใหม่) |
| เลขทะเบียนโรงงาน (เก่า) |
| ประเภทอุตสาหกรรม |
| จังหวัด |
| จำนวนจุดตรวจวัด |
| สถานะคำขอ |
| สถานะ |
| จัดการ |

#### การแสดงผลตามประเภทผู้ใช้งาน

ผู้ประกอบการจะเห็นเฉพาะโรงงานของตัวเองที่เข้าข่ายตามสิทธิ์ `OWN_FACTORY`

เจ้าหน้าที่จะเห็นโรงงานทั้งหมดที่มีการส่งแบบฟอร์มคำขอเข้ามาในระบบ ตาม scope สิทธิ์การมองเห็นข้อมูลที่ได้รับ

#### คอลัมน์จัดการสำหรับเจ้าหน้าที่

ในคอลัมน์ "จัดการ" ของเจ้าหน้าที่มีปุ่มดังนี้

| ปุ่ม | รายละเอียด |
| --- | --- |
| ดูข้อมูล | แสดงรายการและรายละเอียดจุดตรวจวัดที่มีในโรงงานนั้น |
| ดูคำขอ | แสดงรายการและรายละเอียดคำขอต่างๆ ของโรงงานนั้น |
| สถานะ | เปิดเมนูให้เลือกเปลี่ยนสถานะเป็น แสดง หรือ ซ่อน |
| แบบฟอร์มคำขอ | เปิดเมนูให้เลือกแบบฟอร์ม เชื่อมต่อใหม่ เพิ่มจุดตรวจวัด หรือ เพิ่มพารามิเตอร์ |
| ลบ | ลบโรงงานนั้นออกจากระบบ POMS |

#### คอลัมน์จัดการสำหรับผู้ประกอบการ

ในคอลัมน์ "จัดการ" ของผู้ประกอบการมีปุ่มดังนี้

| ปุ่ม | รายละเอียด |
| --- | --- |
| ดูข้อมูล | แสดงรายการและรายละเอียดจุดตรวจวัดที่มีในโรงงานนั้น |
| ดูคำขอ | แสดงรายการและรายละเอียดคำขอต่างๆ ของโรงงานนั้น |
| แบบฟอร์มคำขอ | เปิดเมนูให้เลือกแบบฟอร์ม เชื่อมต่อใหม่ เพิ่มจุดตรวจวัด หรือ เพิ่มพารามิเตอร์ |

เมื่อเลือกแบบฟอร์มคำขอ ให้เปิด bottom sheet ความสูงไม่เกิน header รายละเอียดของแบบฟอร์มจะกำหนดในขั้นตอนถัดไป

### เมนูย่อยรายการคำขอ

เมนูย่อย "รายการคำขอ" แสดงตารางรายการคำขอทั้งหมด โดยประกอบด้วยคอลัมน์ดังนี้

| คอลัมน์ |
| --- |
| ชื่อโรงงาน/บริษัท |
| ประเภทอุตสาหกรรม |
| จังหวัด |
| ประเภท |
| เลขที่คำขอ |
| วันที่ยื่นคำขอ |
| รหัสจุดตรวจวัด |
| วันที่ออกรหัส |
| แบบฟอร์ม |
| สถานะ |
| จัดการ |

#### คอลัมน์จัดการสำหรับเจ้าหน้าที่

ในคอลัมน์ "จัดการ" ของเจ้าหน้าที่มีปุ่ม "ดำเนินการ" เพื่อเปิดดูรายละเอียดคำขอนั้น และดำเนินการตามขั้นตอนต่างๆ

#### คอลัมน์จัดการสำหรับผู้ประกอบการ

ในคอลัมน์ "จัดการ" ของผู้ประกอบการมีปุ่มดังนี้

| ปุ่ม | รายละเอียด |
| --- | --- |
| เปิดดู | แสดงรายละเอียดคำขอเป็น PDF |
| แก้ไข | แก้ไขคำขอเมื่อสถานะอนุญาตให้แก้ไขได้ |
| ยกเลิกคำขอ | ยกเลิกคำขอที่เลือก |

### ขั้นตอนการใช้งาน

1. ผู้ประกอบการเลือกโรงงานที่ต้องการทำธุรกรรมจากตาราง "รายชื่อโรงงาน"
2. ผู้ประกอบการเลือกแบบฟอร์มที่ต้องการ กรอกข้อมูลให้ครบถ้วน และกดส่งแบบฟอร์ม โดยไม่มีการทำ draft สถานะคำขอจะเป็น "รอพิจารณาแบบ"
3. หากเป็นการแก้ไขแบบ ปุ่ม "แก้ไข" จะสามารถกดได้ ผู้ประกอบการต้องแก้ไขคำขอนั้นและส่งให้เจ้าหน้าที่พิจารณาแบบอีกครั้ง หรือสามารถยกเลิกคำขอได้
4. เจ้าหน้าที่เปิดคำขอและตรวจสอบคำขอ เมื่อพิจารณาแบบเรียบร้อย สถานะจะเป็น "รอเชื่อมต่อ" โดยมีระยะเวลาไม่เกิน 30 วัน จำนวนวันให้นับจาก timestamp ที่กดพิจารณา และระบบต้องออกรหัสจุดตรวจวัดแบบ autorun
5. หากเจ้าหน้าที่ตรวจสอบแล้วพบว่าต้องมีการแก้ไข สถานะจะเป็น "รอโรงงานแก้ไข" และให้ผู้ประกอบการดำเนินการตามขั้นตอนการแก้ไขแบบ
6. จากสถานะ "รอเชื่อมต่อ" ผู้ประกอบการต้องเชื่อมต่อข้อมูลจุดตรวจวัดให้ส่งมาที่ระบบ POMS และกดยืนยันการเชื่อมต่อ สถานะจะเป็น "ยืนยันการเชื่อมต่อ"
7. เจ้าหน้าที่ตรวจสอบข้อมูลการเชื่อมต่อ และกดเปลี่ยนสถานะเป็น "เชื่อมต่อแล้ว"

### สถานะคำขอหลัก

| สถานะ | รายละเอียด |
| --- | --- |
| รอพิจารณาแบบ | ผู้ประกอบการส่งแบบฟอร์มแล้ว รอเจ้าหน้าที่ตรวจสอบ |
| รอโรงงานแก้ไข | เจ้าหน้าที่ตรวจสอบแล้วพบว่าต้องให้ผู้ประกอบการแก้ไข |
| รอเชื่อมต่อ | เจ้าหน้าที่พิจารณาแบบเรียบร้อยแล้ว รอผู้ประกอบการเชื่อมต่อข้อมูล ไม่เกิน 30 วัน |
| ยืนยันการเชื่อมต่อ | ผู้ประกอบการเชื่อมต่อข้อมูลและกดยืนยันแล้ว |
| เชื่อมต่อแล้ว | เจ้าหน้าที่ตรวจสอบข้อมูลการเชื่อมต่อและยืนยันเรียบร้อยแล้ว |
